We hebben het onderwerp :has() behandeld

Ik heb mezelf eerst meer verdiept in wat deze code precies doet
Zo heb ik ook geleerd dat het een pseudo-class is en wat dat is.

In de code heb ik de figure:has(figcaption) {} geschreven. Andere hebben hierin ook weer dingen toegevoegd.

In een van de artikelen hadden ze het over siblings.
Ik heb geleerd dat directe siblings worden aangegeven met een + en dat meerdere siblings worden aangegeven met een ~

In de code heb ik een van de afbeeldingen een border gegeven door dit toe te passen.
Dit deed ik met: img:has(+p) {
                   border: .3em solid purple;
                 }

Jeppe heeft me laten zien dat je indent rainbow kan installeren en de code een stuk overzichtelijker is.
Als ik iets vergeet op dezelfde lijn af te sluiten zie je dit veel makkelijker
